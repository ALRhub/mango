MaNGO — Adaptable Graph Network Simulators via Meta-Learning

Conference: NeurIPS 2025

Paper Code arXiv

Abstract

We introduce MaNGO (Meta-learned Adaptable Graph Network Operators), a framework for developing graph network simulators that generalize across diverse physical systems. MaNGO leverages meta-learning to rapidly adapt to unseen domains with minimal fine-tuning, achieving data-efficient generalization in both dynamic and static simulation tasks. We demonstrate that MaNGO outperforms existing GNN-based simulators on multiple physics benchmarks and exhibits strong transfer capabilities across system configurations.

Method

MaNGO combines graph neural simulators with meta-learning to achieve adaptability and data efficiency. The model consists of a graph-based encoder-decoder structure embedded within a meta-learning loop. During training, MaNGO learns to infer simulation dynamics that generalize to new physical systems by optimizing across a distribution of related tasks.

Specifically, MaNGO employs an inner-loop adaptation mechanism that fine-tunes latent dynamics representations to each system, while the outer-loop meta-optimizer updates shared parameters for cross-domain generalization. This design allows MaNGO to adapt to new materials, geometries, or boundary conditions with limited data.
